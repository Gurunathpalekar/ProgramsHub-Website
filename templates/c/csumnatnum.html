C Program to Calculate the Sum of Natural Numbers


In this example, you will learn to calculate the sum of natural numbers entered by the user.
Source code
#include
int main() {
int n, i, sum = 0;
printf("Enter a positive integer: ");
scanf("%d", &n);
for (i = 1; i <= n; ++i) {
sum += i;
}
printf("Sum = %d", sum);
return 0;
}

Output
Enter a positive integer: 100
Sum = 5050
The positive numbers 1, 2, 3... are known as natural numbers. The sum of natural numbers up to 10 is: sum = 1 + 2 + 3 + ... + 10
