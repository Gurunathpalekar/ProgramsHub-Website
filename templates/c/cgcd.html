C Program to Find GCD of two Numbers


In this program calculate GCD of two integers using loops and decision making statements.
Source code
#include
int main()
{
int n1, n2;
printf("Enter two positive integers: ");
scanf("%d %d",&n1,&n2);
while(n1!=n2)
{
if(n1 > n2)
n1 -= n2;
else
n2 -= n1;
}
printf("GCD = %d",n1);
return 0;
}

Output
Enter two positive integers: 81
153
GCD = 9
In this method, smaller integer is subtracted from the larger integer,
and the result is assigned to the variable holding larger integer. This process is continued until n1 and n2 are equal.
